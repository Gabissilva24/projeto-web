O que é Flask?

Flask é um microframework web escrito em Python. Ele é leve, flexível e ideal para o desenvolvimento de aplicações web simples ou sistemas complexos. Diferente de frameworks mais robustos, o Flask oferece apenas o essencial, permitindo que o desenvolvedor escolha as bibliotecas que deseja utilizar.

Principais Vantagens

  • ✔ Simplicidade e fácil aprendizado
  • ✔ Estrutura minimalista
  • ✔ Alta flexibilidade
  • ✔ Integração com bancos de dados
  • ✔ Grande comunidade e documentação

Exemplo Básico de Aplicação

Veja abaixo um exemplo simples de aplicação Flask:

from flask import Flask

app = Flask(__name__)

@app.route("/")
def home():
    return "Olá, mundo!"

if __name__ == "__main__":
    app.run(debug=True)

Neste exemplo, criamos uma aplicação simples que responde à rota principal (/) exibindo uma mensagem no navegador.
